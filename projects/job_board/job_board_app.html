Job Board EU Remote App

About this project

Job Board EU Remote is a complex full-stack application created with Flask, Python, Jinja, HTML, CSS and JavaScript.

While searching for remote jobs in my home country, I noticed that there is a lack of job board platforms that specifically focus on remote job offerings. And I thought that I could just as well make one.

This app is currently under development.

You can follow updates on my LinkedIn Profile.

Responsive design

The responsive design is result of Bootstrap combined with pure CSS.

Where clasic methods were short, I used JavaScript to fill the holes (see example in video).

The design is polished with pleasant colors.

Backend

The core of the backend is database that stores users, their company info and their saved job listings.

With Flask combined with WTForms, users have the flexibility to publish new job listings, edit existing ones, activate and deactivate them.
